# SoundBadge 테마 기획서

## 핵심 방향

현재 "색상만 다른 같은 카드"에서 → **카드 구조/레이아웃 자체가 다른 독립적 템플릿**으로 확장.
사용자가 자기 프로필 성격에 맞는 카드를 골라 쓸 수 있도록 다양한 선택지 제공.

---

## 카테고리 A: 심플/배지 (Simple & Badge)

> GitHub 프로필에 자연스럽게 녹아드는 가벼운 카드.
> 프로필을 깔끔하게 유지하고 싶은 사용자용.

### 1. `badge`

```
┌──────────────────────────────────────────────┐
│ ♪ Now Playing │ Rick Astley - Never Gonna... │
└──────────────────────────────────────────────┘
```

- **컨셉**: shields.io 배지와 동일한 한 줄 인라인 형태
- **구조**: 왼쪽 라벨(배경색) + 오른쪽 텍스트(다른 배경색)
- **크기**: 고정 높이 20~28px, 너비는 텍스트 길이에 따라 가변
- **색상**: 라벨 영역 accent + 텍스트 영역 dark/light
- **특징**: 다른 shields 배지들과 같은 줄에 배치 가능

### 2. `minimal`

```
┌─────────────────────────────────┐
│  🎵  Title Title Title          │
│      Channel Name               │
└─────────────────────────────────┘
```

- **컨셉**: 테두리 하나 + 텍스트만. 썸네일 없음
- **구조**: 좌측 아이콘 + 제목/채널명
- **색상**: 흰 배경, 얇은 회색 테두리, 블랙 텍스트
- **특징**: 가장 가볍고 깨짐 위험 적음. 썸네일 로딩 없어 빠름

### 3. `mono`

```
┌─────────────────────────────────────┐
│  ▶ Rick Astley                      │
│    Never Gonna Give You Up          │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
└─────────────────────────────────────┘
```

- **컨셉**: 흑백. 모노스페이스 폰트 느낌
- **구조**: 채널명 상단, 제목 하단, 얇은 구분선
- **색상**: #0a0a0a 배경, #e5e5e5 텍스트
- **특징**: 개발자 프로필에 가장 잘 어울림. 코드 옆에 놓아도 어색하지 않음

### 4. `clean`

```
┌─────────────────────────────────────────┐
│  ┌──────┐  Title Title Title            │
│  │ Thumb│  Channel Name                 │
│  └──────┘                     ♪         │
└─────────────────────────────────────────┘
```

- **컨셉**: 둥근 모서리, 소프트 그림자, 밝고 깨끗한 느낌
- **구조**: 현재 기본 카드와 유사하지만 더 세련된 비율과 여백
- **색상**: 흰/밝은 회색 배경, 부드러운 그림자
- **특징**: 무난하고 예쁜 기본 선택지

## 카테고리 B: 플레이어 (Player)

> 실제 음악 앱/위젯처럼 보이는 카드.
> "나 이거 듣고 있어"를 직관적으로 전달.

### 5. `stream`

```
┌──────────────────────────────────────────┐
│  ┌────┐  Title Title Title...           │
│  │Art │  Artist Name                    │
│  └────┘                                 │
│  ━━━━━━━━━━━━━━━━━━○━━━━━━━━━━━━━━━━━  │
│  ◂◂    ▶    ▸▸               3:24/4:12 │
└──────────────────────────────────────────┘
```

- **컨셉**: 스트리밍 앱 미니 플레이어 스타일
- **구조**: 앨범아트 + 텍스트 + 프로그레스 바 + 재생 컨트롤(장식)
- **색상**: #1DB954 accent, #191414 배경, 흰 텍스트
- **특징**: 가장 "플레이어"다운 느낌. 컨트롤 버튼은 장식용(클릭 불가)

### 6. `frosted`

```
╭──────────────────────────────────────────╮
│                                          │
│  ┌────────┐  Title Title Title...        │
│  │  Art   │  Artist Name                 │
│  │(둥근)  │                              │
│  └────────┘  ▶  ━━━━━━━○━━━━━━━         │
│                                          │
╰──────────────────────────────────────────╯
```

- **컨셉**: 위젯/Lock Screen 스타일 플레이어
- **구조**: 큰 라운드 앨범아트 + 텍스트 + 슬림 프로그레스바
- **색상**: 앨범아트 dominant color 기반 그라데이션 배경(또는 밝은 톤)
- **특징**: 라운드가 크고 여백이 넓은 위젯 감성

### 7. `dark-player`

```
┌──────────────────────────────────────────┐
│  ┌──────────┐                            │
│  │          │  Title Title Title...      │
│  │   Art    │  Channel Name              │
│  │          │  ▶ 3.2M views              │
│  └──────────┘                            │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
└──────────────────────────────────────────┘
```

- **컨셉**: 다크 테마 동영상 플레이어 UI
- **구조**: 큰 직사각형 썸네일 + 텍스트 + 하단 빨간 프로그레스바
- **색상**: #0f0f0f 배경, #ff0000 accent, 흰 텍스트
- **특징**: 유튜브 링크와 자연스럽게 연결되는 디자인

### 8. `waveform`

```
┌──────────────────────────────────────────┐
│  ┌────┐  Title Title Title...           │
│  │Art │  Artist Name                    │
│  └────┘  ▁▂▃▅▆▇█▇▆▅▃▂▁▂▃▅▇█▇▅▃▂▁▂▃▅  │
└──────────────────────────────────────────┘
```

- **컨셉**: 오디오 파형(waveform) 시각화 UI
- **구조**: 앨범아트 + 텍스트 + 하단에 오디오 파형 시각화(장식)
- **색상**: #ff5500 accent, 흰/밝은 배경
- **특징**: 파형은 랜덤 생성. 역동적이고 음악적인 느낌

## 카테고리 C: 화려한/비주얼 (Visual & Expressive)

> 프로필에서 시선을 확실히 끄는 카드.
> 개성 표현을 중시하는 사용자용.

### 9. `neon`

```
┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  ╔══════╗                                ┃
┃  ║ Thumb ║  ✦ Title Title Title ✦        ┃
┃  ║ glow  ║  Channel Name                 ┃
┃  ╚══════╝                                ┃
┃  ─── ✧ ─── ✧ ─── ✧ ─── ✧ ─── ✧ ────── ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

- **컨셉**: 다크 배경 + 네온 사인 느낌의 글로우 효과
- **구조**: 전체적으로 빛나는 테두리, 썸네일에 글로우, 텍스트에 미세 글로우
- **색상**: 다크 배경, 네온 그린/핑크/시안 중 선택 가능
- **특징**: SVG filter로 glow 효과. 한눈에 시선을 끔

### 10. `gradient-mesh`

- **컨셉**: 여러 색이 섞인 메쉬 그라데이션 배경
- **구조**: 멀티컬러 배경 위에 흰 텍스트
- **색상**: 3~4개 색상 포인트의 메쉬
- **특징**: 매우 트렌디. Instagram 스토리 배경 느낌

### 11. `aurora`

- **컨셉**: 북극 오로라 같은 부드러운 빛의 흐름
- **구조**: 어두운 배경 + 상단에 오로라 띠 효과
- **색상**: 딥 네이비 배경, 그린~퍼플 그라데이션 띠
- **특징**: 은은하면서도 고급스러운 느낌

### 12. `glassmorphism`

```
╭─ · · ─────────────────────────── · · ─╮
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │
│ ░  ┌────┐  Title Title Title      ░░░ │
│ ░  │Art │  Channel Name           ░░░ │
│ ░  └────┘                         ░░░ │
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │
╰─ · · ─────────────────────────── · · ─╯
```

- **컨셉**: 반투명 유리 패널 위에 콘텐츠가 떠 있는 느낌
- **구조**: 반투명 배경 + 미세한 테두리 + 하이라이트 선
- **색상**: rgba 투명 배경, 흰 테두리, 밝은 텍스트
- **특징**: SVG에서 진짜 blur는 불가하므로 noise 텍스쳐 + 투명도로 유사 표현

## 카테고리 D: 레트로/특수 (Retro & Special)

> 독특한 비주얼 컨셉의 카드. 기억에 남는 인상.

### 13. `vinyl`

```
┌─────────────────────────────────────────┐
│      ┌─────────────┐                    │
│      │    ╭───╮    │  Title Title       │
│      │    │Art│    │  Artist Name       │
│      │    ╰───╯    │                    │
│      │  ──────── ◉ │  ♪ My Playlist     │
│      └─────────────┘                    │
└─────────────────────────────────────────┘
```

- **컨셉**: LP 레코드 + 앨범아트가 가운데 있는 디스크
- **구조**: 둥근 디스크(홈 무늬) + 가운데 앨범아트 + 오른쪽 텍스트
- **색상**: 따뜻한 갈색/크림 배경, 검은 디스크
- **특징**: 빈티지 감성. 음악 마니아 느낌

### 14. `cassette`

```
╔══════════════════════════════════════════╗
║  ╔────────────────────────────────────╗  ║
║  ║  Title Title Title...              ║  ║
║  ║  Artist Name        SIDE A  C-60  ║  ║
║  ╚────────────────────────────────────╝  ║
║         ◉━━━━━━━━━━━━━━━━━◉             ║
║        ╭───╮            ╭───╮           ║
║        │   │            │   │           ║
║        ╰───╯            ╰───╯           ║
╚══════════════════════════════════════════╝
```

- **컨셉**: 카세트 테이프 실물을 SVG로 표현
- **구조**: 카세트 외형 + 릴 2개(장식) + 라벨 영역에 텍스트
- **색상**: 크림/베이지 카세트 바디, 갈색 라벨
- **특징**: 레트로 감성 극대화. 디테일한 SVG 일러스트

### 15. `pixel`

```
┌─────────────────────────────────────────┐
│  ████  ▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀          │
│  █  █  Title Title Title                │
│  █  █  Channel Name                     │
│  ████  ♪ ♪ ♪                            │
│  ▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄  │
└─────────────────────────────────────────┘
```

- **컨셉**: 게임보이/픽셀아트 스타일
- **구조**: 픽셀화된 썸네일 + 픽셀 폰트(또는 비트맵 느낌)
- **색상**: 게임보이 그린(#9bbc0f, #8bac0f, #306230, #0f380f)
- **특징**: 픽셀 테두리, 도트 패턴

### 16. `ticket`

```
┌────────────────────────────┬╌╌╌╌┬────────┐
│  ♪ SOUNDBADGE              │    │  ADMIT │
│                            │    │   ONE  │
│  Title Title Title Title   │ ⋮  │        │
│  Artist Name               │ ⋮  │  ♪♪♪   │
│                            │    │        │
│  🎵 pop · classic · 80s   │    │  0001  │
└────────────────────────────┴╌╌╌╌┴────────┘
```

- **컨셉**: 공연 티켓 / 보딩패스 형태
- **구조**: 왼쪽 메인 정보 + 점선 구분 + 오른쪽 스텁
- **색상**: 따뜻한 종이 느낌 배경, 빈티지 잉크
- **특징**: 절취선 효과, 시리얼 넘버(장식)

## 카테고리 E: 텍스트/개발자 (Text & Developer)

> 코드와 함께 놓아도 어색하지 않은 카드.
> 개발자 커뮤니티에 특히 어필.

### 17. `terminal`

```
┌─ 🟢 🟡 🔴 ─── soundbadge ──────────────┐
│  $ now-playing                           │
│  > Title: Rick Astley - Never Gonna...   │
│  > Artist: Rick Astley                   │
│  > Status: ▶ Playing                     │
│  $ █                                     │
└──────────────────────────────────────────┘
```

- **컨셉**: 터미널/쉘 창에 커맨드를 친 결과물처럼
- **구조**: 타이틀바(트래픽 라이트) + 커맨드 라인 형태 텍스트
- **색상**: #1e1e1e 배경, 초록 프롬프트, 흰 텍스트
- **특징**: 모노스페이스 폰트. 개발자가 가장 좋아할 테마

### 18. `editor`

```
┌──────────────────────────────────────────┐
│ 📁 playlist.json  ×  │  🎵 now.ts  ×    │
├──────────────────────────────────────────┤
│  1 │ {                                   │
│  2 │   "title": "Never Gonna Give...",   │
│  3 │   "artist": "Rick Astley",          │
│  4 │   "status": "playing ▶"             │
│  5 │ }                                   │
└──────────────────────────────────────────┘
```

- **컨셉**: 코드 에디터 패널처럼 코드로 표현
- **구조**: 탭 바 + 라인 넘버 + JSON/코드 형태의 정보
- **색상**: 에디터 다크 테마 (#1e1e1e, #569cd6, #ce9178, #d4d4d4)
- **특징**: 실제 코드 에디터의 syntax highlighting 재현

### 19. `markdown`

```
┌──────────────────────────────────────────┐
│  ## 🎵 Now Playing                       │
│                                          │
│  **Rick Astley** - *Never Gonna Give...*│
│                                          │
│  `pop` `classic` `80s`                   │
│  ─────────────────────────────────────── │
└──────────────────────────────────────────┘
```

- **컨셉**: 마크다운 렌더링된 문서처럼
- **구조**: 마크다운 헤딩, 볼드, 이탤릭, 코드 스팬 시각적 표현
- **색상**: 흰 배경, GitHub 마크다운 스타일 서체/색상
- **특징**: GitHub README 안에서 가장 자연스럽게 어울림

## 카테고리 F: 한국어/문화 (Korean & Cultural)

> 한국어 라벨과 한국 음악 서비스 감성의 카드.
> 한국 개발자 커뮤니티 특화.

### 20. `k-chart`

```
┌──────────────────────────────────────────┐
│  01  ┌────┐  Title Title Title...        │
│  ▲2  │Art │  Artist Name                 │
│      └────┘  좋아요 ♥ 12.3만              │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
└──────────────────────────────────────────┘
```

- **컨셉**: 한국 음악 차트 리스트 아이템 스타일
- **구조**: 순위 번호(장식) + 앨범아트 + 제목/아티스트 + 좋아요(장식)
- **색상**: 그린 accent, 밝은 배경
- **특징**: 한국 음악 서비스 차트에 익숙한 사용자에게 친숙한 레이아웃
- **한국어**: "좋아요", "지금 재생 중" 라벨
- **주의**: 특정 서비스 UI를 그대로 복제하지 않고, 차트 리스트라는 공통 패턴만 참고

### 21. `karaoke`

```
┌──────────────────────────────────────────┐
│ ✦ ˚ · .   ·  ✦  . ·  ˚  ✦ · .  ✦  ˚  │
│                                          │
│     🎤 Title Title Title Title 🎤       │
│         ~~~ Artist Name ~~~              │
│                                          │
│ . ✦  ˚ ·  .  ✦ ·  ˚ . ✦  · ˚  .  ✦ · │
└──────────────────────────────────────────┘
```

- **컨셉**: 노래방 화면의 자막 느낌
- **구조**: 화려한 배경 + 중앙 정렬 큰 텍스트 + 반짝이 장식
- **색상**: 진한 파랑/보라 그라데이션 배경, 노란 금색 텍스트
- **특징**: 텍스트에 그라데이션/글로우 효과. 화려하고 재밌는 느낌
- **한국어**: "♪ 지금 부르는 노래" 라벨

### 22. `hangul`

```
┌──────────────────────────────────────────┐
│                                          │
│  지금 듣는 중 ♪                           │
│                                          │
│  Title Title Title Title                 │
│  ── Artist Name                          │
│                                          │
└──────────────────────────────────────────┘
```

- **컨셉**: 한국어 타이포그래피 중심. 썸네일 없이 텍스트로 승부
- **구조**: 큰 한국어 라벨 + 제목 + 아티스트
- **색상**: 미니멀 (흰/검정 또는 파스텔)
- **특징**: 한글 자체의 아름다움을 살리는 디자인. 다양한 한국어 라벨 옵션
- **한국어 라벨 옵션**: "지금 듣는 중", "내 플레이리스트", "오늘의 음악", "반복 재생 중"

## 아키텍처 노트

현재 `renderCardSvg()` 하나로 모든 테마를 처리하는 구조에서,
**테마(템플릿)별 독립 렌더러**로 전환 필요.

```
src/templates/
├── badge.ts         # renderBadge(options): string
├── clean.ts         # renderClean(options): string
├── stream.ts        # renderStream(options): string
├── terminal.ts      # renderTerminal(options): string
├── neon.ts          # renderNeon(options): string
├── ...
└── index.ts         # getRenderer(templateName) → render function
```

각 렌더러는 동일한 `CardOptions`를 받아 SVG 문자열을 반환.
색상 토큰은 각 렌더러 내에서 자체 정의 (또는 variant 옵션으로 제공).

```typescript
// API 호출 예시
GET /api/card.svg?url=YOUTUBE_URL&template=stream
GET /api/card.svg?url=YOUTUBE_URL&template=terminal
GET /api/card.svg?url=YOUTUBE_URL&template=badge
```

기존 `theme` 파라미터 → `template` 파라미터로 변경하거나,
`template`이 카드 구조, `variant`가 색상 변형 역할로 분리 가능.
